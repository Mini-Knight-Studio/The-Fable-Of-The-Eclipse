Loopie Engine

The foundation of the entire project, Loopie is a custom, from scratch game engine designed to bring our game to life. Builded using multiple external libraries to maximize development speed, its architecture was designed to mimic Unity, ensuring a smooth and familiar learning curve for the team.

Key Features

Core Systems: Robust Scene/Entity architecture, an Asset & Resource management system, and an intuitive, user friendly editor.

Graphics & Visuals: Advanced lighting, custom shader support, a dedicated Particle System, and an Animation System featuring skinned mesh support.

Gameplay & Audio: Integrated Audio System, built-in basic collision detection, and a flexible UI system.

Scripting & Workflow: Full C# scripting integration, project management tools, and a semi-automatic game deployment pipeline.

Scene Data Extractor

To overcome our engine's initial lack of a native prefab system, we designed a clever workaround. We used AI to generate this external utility tool based on our specifications. The SceneDataExtractor allowed us to seamlessly move, clone, delete, and preview entities across different scenes, effectively solving a critical workflow gap and saving us massive amounts of development time.
